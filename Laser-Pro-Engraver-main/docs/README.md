## In-App Help

LaserTrace Pro includes comprehensive in-app documentation:

### Interactive Tutorial
- Step-by-step guided tour for new users
- Covers: image loading, layers, settings, path generation, engraving
- Skippable and can be re-enabled in settings

### Searchable Glossary
- 30+ laser engraving terms with definitions
- Organized by category: basics, settings, features, safety
- Includes practical examples
- Search functionality for quick reference

### Help Guide
- Quick reference documentation
- Driver installation instructions
- Workflow essentials
- Technical parameter guide
- Safety protocols

### Custom Presets Manager
- Save/load machine configurations
- Material-specific settings
- Import/export presets
